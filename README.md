# Echo-server

Данное приложение используется для проверки корректной работы того или иного сервера. Для этого надо запустить данный билд на сервере и 
при необходимости отладки можно проверить работает ли данный сервер. На указанный порт отправте любую строку, сообщение и в случае
если сервер работает, то он вам вернет туже самую строку обратно. 

Ссылка на источкик: https://www.rfc-editor.org/rfc/rfc862

Запуск echo-server (macOS):
1. `git clone https://github.com/SidorinAA/echo-server.git`
2. `gradle build` (перейдите в папку проекта)
3. `docker build -t echo-server .`
4. `docker run -d -p 7:7 echo-server`

Для отправки сообщения введите:
``` 
echo "Hello World" | nc localhost 7
```
Чтобы остановить сервер введите 
```
echo "stop" | nc localhost 7
```
